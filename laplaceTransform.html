Comprehensive Laplace Transform Guide

This guide provides a thorough treatment of Laplace transforms with detailed examples, proofs, and applications for Applied Mathematics 3.

1. Introduction to Laplace Transform

Definition (Laplace Transform) The Laplace transform of a function \( f(t) \), defined for all real numbers \( t \geq 0 \), is the function \( F(s) \), defined by: \[ \mathcal{L}\{f(t)\} = F(s) = \int_{0}^{\infty} e^{-st} f(t) \, dt \] where \( s \) is a complex number frequency parameter \( s = \sigma + i\omega \).
Example 1.1: Laplace Transform of a Constant Function Find \( \mathcal{L}\{1\} \).
1 Apply the definition: \[ \mathcal{L}\{1\} = \int_{0}^{\infty} e^{-st} \cdot 1 \, dt \]
2 Evaluate the improper integral: \[ = \lim_{b \to \infty} \int_{0}^{b} e^{-st} \, dt = \lim_{b \to \infty} \left[ -\frac{1}{s} e^{-st} \right]_0^b \]
3 Compute the limit: \[ = \lim_{b \to \infty} \left( -\frac{1}{s} e^{-sb} + \frac{1}{s} e^{0} \right) = 0 + \frac{1}{s} = \frac{1}{s} \] provided \( \text{Re}(s) > 0 \) for convergence.

Conclusion: \( \mathcal{L}\{1\} = \frac{1}{s} \) for \( \text{Re}(s) > 0 \).

Example 1.2: Laplace Transform of an Exponential Function Find \( \mathcal{L}\{e^{at}\} \).
1 Apply the definition: \[ \mathcal{L}\{e^{at}\} = \int_{0}^{\infty} e^{-st} e^{at} \, dt = \int_{0}^{\infty} e^{-(s-a)t} \, dt \]
2 Evaluate the integral: \[ = \lim_{b \to \infty} \left[ -\frac{1}{s-a} e^{-(s-a)t} \right]_0^b \]
3 Compute the limit: \[ = \lim_{b \to \infty} \left( -\frac{1}{s-a} e^{-(s-a)b} + \frac{1}{s-a} \right) \] The first term approaches 0 if \( \text{Re}(s) > \text{Re}(a) \).

Conclusion: \( \mathcal{L}\{e^{at}\} = \frac{1}{s-a} \) for \( \text{Re}(s) > \text{Re}(a) \).

2. Existence Conditions and Properties

Theorem 2.1 (Existence of Laplace Transform) If \( f(t) \) satisfies:
  1. Piecewise Continuity: \( f(t) \) is piecewise continuous on every finite interval \( 0 \leq t \leq T \)
  2. Exponential Order: There exist constants \( M > 0 \), \( a \), and \( N > 0 \) such that \( |f(t)| \leq Me^{at} \) for all \( t \geq N \)
then the Laplace transform \( F(s) \) exists for all \( \text{Re}(s) > a \).
Proof:
1 Split the integral: \[ \int_{0}^{\infty} |e^{-st}f(t)| dt = \int_{0}^{N} |e^{-st}f(t)| dt + \int_{N}^{\infty} |e^{-st}f(t)| dt \]
2 First integral is finite since \( f \) is piecewise continuous on \( [0,N] \) and \( e^{-st} \) is continuous.
3 For the second integral, using the exponential order condition: \[ \int_{N}^{\infty} |e^{-st}f(t)| dt \leq M \int_{N}^{\infty} e^{-(s-a)t} dt \]
4 This converges when \( \text{Re}(s-a) > 0 \), i.e., \( \text{Re}(s) > a \).
Note: These conditions are sufficient but not necessary. Some functions that don't satisfy these conditions may still have Laplace transforms.
Theorem 2.2 (Linearity Property) The Laplace transform is a linear operator: \[ \mathcal{L}\{af(t) + bg(t)\} = a\mathcal{L}\{f(t)\} + b\mathcal{L}\{g(t)\} = aF(s) + bG(s) \] for any constants \( a, b \) and functions \( f, g \) whose Laplace transforms exist.
Example 2.1: Using Linearity Property Find \( \mathcal{L}\{3t^2 - 4\sin(5t) + 6e^{-2t}\} \).
1 Apply linearity: \[ = 3\mathcal{L}\{t^2\} - 4\mathcal{L}\{\sin(5t)\} + 6\mathcal{L}\{e^{-2t}\} \]
2 Use known transforms: \[ \mathcal{L}\{t^2\} = \frac{2!}{s^3} = \frac{2}{s^3} \] \[ \mathcal{L}\{\sin(5t)\} = \frac{5}{s^2 + 25} \] \[ \mathcal{L}\{e^{-2t}\} = \frac{1}{s + 2} \]
3 Combine results: \[ = 3\left(\frac{2}{s^3}\right) - 4\left(\frac{5}{s^2 + 25}\right) + 6\left(\frac{1}{s + 2}\right) \] \[ = \frac{6}{s^3} - \frac{20}{s^2 + 25} + \frac{6}{s + 2} \]

3. Transform of Derivatives

Theorem 3.1 (First Derivative) If \( f(t) \) is continuous and of exponential order, and \( f'(t) \) is piecewise continuous on \( [0, \infty) \), then: \[ \mathcal{L}\{f'(t)\} = sF(s) - f(0) \]
Proof:
1 Start with the definition: \[ \mathcal{L}\{f'(t)\} = \int_{0}^{\infty} e^{-st} f'(t) dt \]
2 Integrate by parts: \( u = e^{-st} \), \( dv = f'(t) dt \) \[ = \left. e^{-st}f(t) \right|_0^\infty + s \int_0^\infty e^{-st} f(t) dt \]
3 Evaluate boundary terms: \[ = \lim_{t \to \infty} e^{-st}f(t) - e^{0}f(0) + sF(s) \] The first term vanishes for \( \text{Re}(s) > a \) (from exponential order).
4 Final result: \[ = -f(0) + sF(s) \]
Theorem 3.2 (Higher Order Derivatives) For \( n^{th} \) derivative: \[ \mathcal{L}\{f^{(n)}(t)\} = s^n F(s) - s^{n-1}f(0) - s^{n-2}f'(0) - \cdots - f^{(n-1)}(0) \]
Example 3.1: Solving a Differential Equation Solve \( y'' + 3y' + 2y = 0 \) with \( y(0) = 1 \), \( y'(0) = 0 \).
1 Take Laplace transform of both sides: \[ \mathcal{L}\{y''\} + 3\mathcal{L}\{y'\} + 2\mathcal{L}\{y\} = 0 \]
2 Apply derivative transforms: \[ [s^2Y(s) - sy(0) - y'(0)] + 3[sY(s) - y(0)] + 2Y(s) = 0 \]
3 Substitute initial conditions: \[ s^2Y(s) - s + 3sY(s) - 3 + 2Y(s) = 0 \]
4 Solve for \( Y(s) \): \[ (s^2 + 3s + 2)Y(s) = s + 3 \] \[ Y(s) = \frac{s + 3}{s^2 + 3s + 2} \]
5 Factor denominator and partial fractions: \[ \frac{s + 3}{(s + 1)(s + 2)} = \frac{A}{s + 1} + \frac{B}{s + 2} \] Solving gives \( A = 2 \), \( B = -1 \)
6 Take inverse transform: \[ y(t) = 2e^{-t} - e^{-2t} \]

4. Shifting Theorems

Theorem 4.1 (First Shifting Theorem - Frequency Shift) If \( \mathcal{L}\{f(t)\} = F(s) \), then: \[ \mathcal{L}\{e^{at}f(t)\} = F(s - a) \]
Proof: \[ \mathcal{L}\{e^{at}f(t)\} = \int_0^\infty e^{-st} e^{at} f(t) dt = \int_0^\infty e^{-(s-a)t} f(t) dt = F(s - a) \]
Example 4.1: Using First Shifting Theorem Find \( \mathcal{L}\{e^{3t}\sin(4t)\} \).
1 We know \( \mathcal{L}\{\sin(4t)\} = \frac{4}{s^2 + 16} \)
2 Apply the first shifting theorem with \( a = 3 \): \[ \mathcal{L}\{e^{3t}\sin(4t)\} = \frac{4}{(s - 3)^2 + 16} \]
Theorem 4.2 (Second Shifting Theorem - Time Shift) If \( \mathcal{L}\{f(t)\} = F(s) \) and \( a > 0 \), then: \[ \mathcal{L}\{f(t - a)u(t - a)\} = e^{-as}F(s) \] where \( u(t) \) is the unit step function.
Example 4.2: Using Second Shifting Theorem Find \( \mathcal{L}\{(t - 2)^3 u(t - 2)\} \).
1 We know \( \mathcal{L}\{t^3\} = \frac{6}{s^4} \)
2 Apply the second shifting theorem with \( a = 2 \): \[ \mathcal{L}\{(t - 2)^3 u(t - 2)\} = e^{-2s} \frac{6}{s^4} \]

5. Convolution Theorem

Definition (Convolution) The convolution of two functions \( f \) and \( g \) is defined as: \[ (f * g)(t) = \int_0^t f(\tau)g(t - \tau) d\tau \]
Theorem 5.1 (Convolution Theorem) \[ \mathcal{L}\{f(t) * g(t)\} = F(s)G(s) \] Equivalently: \[ \mathcal{L}^{-1}\{F(s)G(s)\} = f(t) * g(t) \]
Example 5.1: Using Convolution Theorem Find \( \mathcal{L}^{-1}\left\{\frac{1}{s(s^2 + 1)}\right\} \).
1 Recognize as product: \[ \frac{1}{s(s^2 + 1)} = \frac{1}{s} \cdot \frac{1}{s^2 + 1} \]
2 Identify inverse transforms: \[ \mathcal{L}^{-1}\left\{\frac{1}{s}\right\} = 1 \] \[ \mathcal{L}^{-1}\left\{\frac{1}{s^2 + 1}\right\} = \sin t \]
3 Apply convolution theorem: \[ = 1 * \sin t = \int_0^t 1 \cdot \sin(t - \tau) d\tau \]
4 Evaluate the integral: \[ = \left. \cos(t - \tau) \right|_0^t = \cos 0 - \cos t = 1 - \cos t \]

6. Applications to Differential Equations

Example 6.1: Nonhomogeneous Equation Solve \( y'' + y = \sin t \) with \( y(0) = 1 \), \( y'(0) = 0 \).
1 Take Laplace transform: \[ s^2Y(s) - s + Y(s) = \frac{1}{s^2 + 1} \]
2 Solve for \( Y(s) \): \[ (s^2 + 1)Y(s) = \frac{1}{s^2 + 1} + s \] \[ Y(s) = \frac{1}{(s^2 + 1)^2} + \frac{s}{s^2 + 1} \]
3 Take inverse transforms: \[ \mathcal{L}^{-1}\left\{\frac{s}{s^2 + 1}\right\} = \cos t \] For the first term, use convolution: \[ \frac{1}{(s^2 + 1)^2} = \frac{1}{s^2 + 1} \cdot \frac{1}{s^2 + 1} \] \[ \mathcal{L}^{-1}\left\{\frac{1}{(s^2 + 1)^2}\right\} = \sin t * \sin t \]
4 Compute convolution: \[ \int_0^t \sin \tau \sin(t - \tau) d\tau = \frac{1}{2}(\sin t - t \cos t) \]
5 Final solution: \[ y(t) = \frac{1}{2}(\sin t - t \cos t) + \cos t = \frac{1}{2}\sin t + \left(1 - \frac{t}{2}\right)\cos t \]

7. Laplace Transform of Periodic Functions

Theorem 7.1 (Periodic Functions) If \( f(t) \) is periodic with period \( T \) (i.e., \( f(t + T) = f(t) \) for all \( t \geq 0 \)), then: \[ \mathcal{L}\{f(t)\} = \frac{1}{1 - e^{-sT}} \int_0^T e^{-st} f(t) dt \]
Example 7.1: Square Wave Find the Laplace transform of the square wave with period \( 2a \) defined by: \[ f(t) = \begin{cases} 1 & 0 \leq t < a \\ 0 & a \leq t < 2a \end{cases} \] and \( f(t + 2a) = f(t) \) for all \( t \geq 0 \).
1 Compute integral over one period: \[ \int_0^{2a} e^{-st} f(t) dt = \int_0^a e^{-st} dt = \frac{1 - e^{-as}}{s} \]
2 Apply periodic function theorem: \[ \mathcal{L}\{f(t)\} = \frac{1 - e^{-as}}{s(1 - e^{-2as})} = \frac{1}{s(1 + e^{-as})} \]

8. Impulse Functions and Dirac Delta

Definition (Unit Impulse Function) The Dirac delta function \( \delta(t - a) \) is defined by: \[ \delta(t - a) = 0 \text{ for } t \neq a \] \[ \int_{-\infty}^\infty \delta(t - a) dt = 1 \] and for any continuous function \( f \): \[ \int_{-\infty}^\infty f(t) \delta(t - a) dt = f(a) \]
Theorem 8.1 (Laplace Transform of Delta Function) \[ \mathcal{L}\{\delta(t - a)\} = e^{-as} \] In particular, \( \mathcal{L}\{\delta(t)\} = 1 \).
Example 8.1: Differential Equation with Impulse Solve \( y'' + 2y' + 2y = \delta(t - \pi) \) with \( y(0) = 1 \), \( y'(0) = 0 \).
1 Take Laplace transform: \[ s^2Y(s) - s + 2sY(s) - 2 + 2Y(s) = e^{-\pi s} \]
2 Solve for \( Y(s) \): \[ (s^2 + 2s + 2)Y(s) = e^{-\pi s} + s + 2 \] \[ Y(s) = \frac{e^{-\pi s}}{(s + 1)^2 + 1} + \frac{s + 2}{(s + 1)^2 + 1} \]
3 Take inverse transforms: \[ \mathcal{L}^{-1}\left\{\frac{s + 2}{(s + 1)^2 + 1}\right\} = e^{-t}(\cos t + \sin t) \] \[ \mathcal{L}^{-1}\left\{\frac{e^{-\pi s}}{(s + 1)^2 + 1}\right\} = u(t - \pi)e^{-(t - \pi)}\sin(t - \pi) \]
4 Final solution: \[ y(t) = e^{-t}(\cos t + \sin t) + u(t - \pi)e^{-(t - \pi)}\sin(t - \pi) \]

10. Detailed Solutions

Problem 1 Solution

1 Use multiplication by \( t^n \) property: \[ \mathcal{L}\{t^2 \cos(4t)\} = \frac{d^2}{ds^2} \left( \frac{s}{s^2 + 16} \right) \]
2 Compute derivatives: \[ \frac{d}{ds} \left( \frac{s}{s^2 + 16} \right) = \frac{(s^2 + 16) - s(2s)}{(s^2 + 16)^2} = \frac{16 - s^2}{(s^2 + 16)^2} \] \[ \frac{d^2}{ds^2} \left( \frac{s}{s^2 + 16} \right) = \frac{( -2s)(s^2 + 16)^2 - (16 - s^2)(2)(s^2 + 16)(2s)}{(s^2 + 16)^4} \] \[ = \frac{2s^3 - 96s}{(s^2 + 16)^3} \]
3 Apply first shifting theorem (\( a = -3 \)): \[ \mathcal{L}\{t^2 e^{-3t} \cos(4t)\} = \left. \frac{2s^3 - 96s}{(s^2 + 16)^3} \right|_{s \to s + 3} \] \[ = \frac{2(s + 3)^3 - 96(s + 3)}{((s + 3)^2 + 16)^3} \]

Problem 3 Solution

1 Take Laplace transform: \[ s^3Y(s) - Y(s) = \frac{1}{s - 2} \]
2 Solve for \( Y(s) \): \[ Y(s) = \frac{1}{(s^3 - 1)(s - 2)} = \frac{1}{(s - 1)(s^2 + s + 1)(s - 2)} \]
3 Partial fraction decomposition: \[ Y(s) = \frac{A}{s - 1} + \frac{Bs + C}{s^2 + s + 1} + \frac{D}{s - 2} \] Solving gives \( A = -\frac{1}{3} \), \( B = 0 \), \( C = -\frac{1}{3} \), \( D = \frac{1}{7} \)
4 Complete the square and take inverse transforms: \[ y(t) = -\frac{1}{3}e^t - \frac{1}{3}e^{-t/2}\cos\left(\frac{\sqrt{3}}{2}t\right) + \frac{1}{7}e^{2t} \]
